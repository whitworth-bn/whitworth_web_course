Definitions

HTML
Hypertext Markup Language, the standered language used to create webpages.
css
Cascading Style Sheets,CSS is a computer language for laying out and structuring web pages (HTML or XML).
Elements
Elements are the parts of a webpage,including text, images, and interactive components.
Tags
Tags refer to the codes used in HTML to define different parts of webpages, like headings, paragraphs, images, or links.
Whitespace Collapse
The browser's behavior of automatically reducing multipule consecutive space,tabs, or line brreakes in HTML code to just a single space when rendering the webpage.
Structural Markup
A way of using HTML tags to indicate the structure and meaning of content on a web page.
Semantic Markup
Using HTML tags that convey the meaning or "semantics" of the content within them
